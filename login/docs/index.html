What is This?

The miniPAN is a PHP based file management script, that is seamlessly integrated with TinyMCE, CKEditor. This script can also be used standalone and it's easy to integrate with your backend system.

Features

  • Easy to integrate, seamless integration with WYSIWYG editors (TinyMCE, CKEditor)
  • Create directories, rename, delete instantly
  • Upload files, allow/deny file types
  • Ajax based interface using a JSON bridge.
  • Multiple Upload
  • Browse or Dropable Upload option
  • Rename, Delete, Move File Function
  • Linkage Different Platforms, WYSIWYG editors or Your Own Form Fields
  • Linkage HTML link or image view and text url
  • 28 File Icons
  • Folder Statistics (File Count & Size)
  • Sorting & Listing Module
  • Customizable Bootstarp Theme
  • Language support

Installation

The miniPAN is a ready to use system. It is no required database connection.

Basically miniPAN; if you don't use existing main storage folder, it's create main folder resources named to your root directory.

Please check out Configuration area for folder, allowed files settings.

Configuration

Define Main Folder

open: [miniPan]/configs.php

$pan_stock_folder = "resources";

This variable will define of your main storage folder, if it's not exists on your web server, it's will be created automatically.

Uploading Settings

open: [miniPan]/configs.php

Safe Name

define('pan_set_safename',1);

Uploaded file names is contain invalid characters, it's will be replaced with _ character.

If you wanna close this option, change variable value to "0"

Mime Check

define('pan_set_mimecheck',1);

If the class check the MIME against the allowed list. Allowed file type list linked below.

If you wanna close this option, change variable value to "0"

Overwrite

define('pan_set_overwrite',0);

If uploaded file already exists on your selected directory, uploading event will give you an error. (if auto rename option is true, it's will rename uploaded file name)

If you wanna allow overwrite new file on existing file, change variable value to "1"

Auto Rename

define('pan_set_autorename',1);

If uploaded file already exists on your selected directory,  it's will rename uploaded file name. (e.g. file.jpg, file_1.jpg, file_2.jpg)

If you wanna close this option, change variable value to "0"

Maximum Uploaded File Size

define('pan_set_max_file_size','2097152');

This variable define maximum uploaded file size. It is must be byte value. Default is defined for 2 MB.

If you do not know to calculate storage sizes, you can visit Google's unit converter solution: http://bit.ly/1kbiRBI

Allowed File Types

$upl_allow_files = array('image/gif','image/jpeg','image/jpeg','image/png');

If you use multi-user system, this will give you administrate of uploaded file types. This variables contain MIME TYPES of your allowed files.

image/gif = GIF file
image/jpeg = JPEG file

You can check all file mime-types on this site: http://www.iana.org/assignments/media-types/

Basic Form

miniPAN uses HTML5 data- attributes;

data-pan-model - (normal / fancybox) This define miniPAN opener.
data-pan-field - This define of your link field
data-pan-link - (default / link) This is define link type. default callback to HTML codes, links callback to only text file url.
data-pan-platform - (normal / tinymce / ckeditor) If you use different form fields, this will generate platform script.

<input name="text_field" type="text" id="text_field" size="40">

<button data-pan-model="normal" data-pan-field="text_field" data-pan-link="link" data-pan-platform="normal" class="minipan" type="button">miniPan</button>

TinyMCE

Add these configs on your TinyMCE config file. Fancybox opener URL can be change with your miniPAN source.

IMPORTANT: Don't forget to change selector ID on miniPAN URL.

            <textarea name="text_area" id="text_area"></textarea>
            <!-- CALL TINYMCE CONFIGS -->
            <script src="tinymce/js/tinymce/tinymce.min.js"></script>
            <script>
                    tinymce.init({
                            selector:'#text_area',
                            height : 500,
                            menubar:false,
                            relative_urls: false,
                            remove_script_host: false,
                            plugins: 'textcolor,link,image,charmap,code,table,emoticons',
                            toolbar1: 'formatselect fontselect fontsizeselect forecolor backcolor',
                            toolbar2: 'bold italic underline strikethrough alignleft aligncenter alignright alignjustify bullist numlist outdent indent blockquote',
                            toolbar3: 'link unlink image charmap | table | code | minipan',
                            setup: function(editor) {
                                editor.addButton('minipan', {
                                    addClass: 'minipan',
                                    text: 'miniPAN',
                                    icon: false,
                                    onclick: function(e) {
                                        // Fancybox Example
                                        $.fancybox({
                                            autoSize : true,
                                            type     : 'iframe',
                                            href     : '../index.php?pf=' + editor.id + '&pm=default&pp=tinymce&o=fancybox'
                                        });
                                        // ******
                                    }
                                });
                            }                        
                        });
            </script>
            <!-- CALL TINYMCE CONFIGS -->

CKEditor

CKEditor uses own fileBrowser settings, change browser URL to your own miniPAN source.

IMPORTANT: Don't forget to change selector ID on miniPAN URL.

            <textarea name="text_area" id="text_area"></textarea>
            <!-- CALL CKEDITOR CONFIGS -->
            <script src="ckeditor/ckeditor.js"></script>
            <script>
                CKEDITOR.replace( 'text_area' ,{
                    filebrowserBrowseUrl: '../index.php?pf=text_area&pm=default&pp=ckeditor&o=normal',
                    filebrowserImageBrowseUrl: '../index.php?pf=text_area&pm=default&pp=ckeditor&o=normal&type=Images',
                    filebrowserWindowWidth: '820',
                    filebrowserWindowHeight: '590'
                    });
            </script>
            <!-- CALL CKEDITOR CONFIGS -->

 

 

Advanced Settings

Multiple Language

miniPAN contain only English language. If you wanna add more language, it is can modify on;

[miniPAN]/lang/pan-en.php

Languages calling inside of config.php

Change Default Language;

open: [miniPAN]/lang/pan-en.php

Replace values with your own language, like;

define('minipan_browse','Browse');

to

define('minipan_browse','Navegar');

Add More Language

Languages calling inside of config.php

You can use Cookie/Session system on language files;

$default_lang = "en";

if(!isset($_COOKIE['panLang'])){$lang=$default_lang;}else{$lang=$_COOKIE['panLang'];}

include_once('lang/pan-'. $lang .'.php'); # Language variables
